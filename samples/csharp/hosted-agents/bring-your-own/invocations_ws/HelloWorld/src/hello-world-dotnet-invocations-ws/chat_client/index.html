🎙️ Voice Live Hello World — standalone client

Browsers can't set the Foundry Authorization header. To talk to a hosted agent from this page, run the bundled local proxy in another terminal: 
dotnet run --project chat_client/Proxy -- --foundry https://<account>.services.ai.azure.com/api/projects/<project> --agent hello-world-dotnet-invocations-ws
then point the WebSocket URL below at ws://localhost:8765/invocations_ws. The proxy injects Authorization: Bearer <az token> on your behalf. Direct wss://*.azure.com URLs are blocked here because the browser can't authenticate them.
