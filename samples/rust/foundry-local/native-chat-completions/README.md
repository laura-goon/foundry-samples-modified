# Native Chat Completions (Rust)

Non-streaming and streaming chat completions using the native chat client.

> Part of the [Foundry Local samples for Rust](../).

## Prerequisites

- [Foundry Local](https://learn.microsoft.com/azure/foundry-local/) installed
- Rust 1.70+

## Run

```bash
cargo run
```

See the [Foundry Local documentation](https://learn.microsoft.com/azure/foundry-local/) for more details.
